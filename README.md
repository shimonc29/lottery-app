# מערכת הגרלה — ערוץ + שמירת מספר + טופס

אפליקציה קטנה ועצמאית, **ללא שום חיבור לוואטסאפ** (אפס סיכון חסימה).
המשתתף עובר 3 שלבים (מעקב אחרי ערוץ → שמירת המספר → צילום מסך) וממלא טופס.
המנהל רואה את כל הנרשמים, מגריל זוכה, ומייצא את כולם כקובץ אנשי קשר לייבוא בלחיצה לטלפון.

## הקמה ב-Replit (5 דקות)

1. צור Repl חדש מסוג **Node.js** והעלה את כל הקבצים (או חבר ל-GitHub).
2. ב-Shell הרץ: `npm install`
3. פתח את לשונית **Secrets** (🔒) והגדר את המשתנים הבאים:

| מפתח | משמעות | דוגמה |
|---|---|---|
| `BUSINESS_NAME` | שם העסק | `צמרת` |
| `PRIZE_TEXT` | תיאור הפרס | `שובר על 1,000 ₪` |
| `CHANNEL_URL` | קישור לערוץ הוואטסאפ | `https://whatsapp.com/channel/XXXX` |
| `CLIENT_NAME` | השם שיישמר באנשי הקשר | `דני - צמרת` |
| `CLIENT_PHONE` | מספר הלקוח, **פורמט בינלאומי ללא +** | `972521234567` |
| `CONTACT_PREFIX` | קידומת לשם בייצוא (לניקוי קל אח״כ) | `הגרלה` |
| `ADMIN_USER` | שם משתמש לאדמין | `admin` |
| `ADMIN_PASSWORD` | **סיסמה — חובה לשנות!** | `סיסמה-חזקה` |

4. לחץ **Run**. הדף יעלה.

## כתובות

- `/` — דף ההגרלה (זה הקישור שמפרסמים בסרטון/בסטטוס).
- `/admin` — דאשבורד ניהול (דורש שם משתמש + סיסמה).

## מה עושים בדאשבורד

- רואים את כל המשתתפים ומספר הנרשמים.
- **🎲 הגרלת זוכה** — בוחר משתתף אקראי.
- **📇 ייצוא אנשי קשר (vCard)** — מוריד קובץ `.vcf` עם כל המספרים.
  שולחים אותו לטלפון של הלקוח ופותחים → כל אנשי הקשר נשמרים בבת אחת.
  *(זה מה שסוגר את ההדדיות: הלקוח שומר אותם → הם יראו את הסטטוס שלו.)*
- **📊 ייצוא לאקסל (CSV)** — לגיבוי/ניתוח.

## הערות

- האחסון הוא קובץ `data.json` מקומי. ב-Replit מומלץ לוודא שהוא נשמר (Always On / DB).
  להחלפה ל-PostgreSQL בעתיד צריך לשנות רק את `readEntries` / `writeEntries` ב-`server.js`.
- אין אימות אמיתי שמישהו "באמת" עקב/שמר — זה על בסיס אמון (כמו בכל מערכת כזו).
  רשת הביטחון: גם מי שלא עקב מקבל את התוכן דרך הערוץ עצמו.
- כדאי לצרף בטופס אישור קבלת עדכונים (כבר קיים) — שמירה על חוק הספאם.
